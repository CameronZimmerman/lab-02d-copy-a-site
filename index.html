Welcome to Alchemy's Foundations Prep!

We’re delighted to have you as a member of our Foundations Prep Software Development course, where you'll learn the basics of computer programming. It'll be a lot of work and a lot of fun. The emphasis is on getting people set up with real developer tools, and going beyond the “introductory” nature of online tutorials to show you real developer workflows for managing code and how JavaScript, HTML, and CSS go together to make web apps.
